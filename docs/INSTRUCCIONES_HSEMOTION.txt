SISTEMA DE ANÁLISIS COGNITIVO CON HSEMOTION
===========================================

INSTALACIÓN
-----------

1. Crear entorno virtual:
   Windows: python -m venv venv && venv\Scripts\activate
   Linux/Mac: python -m venv venv && source venv/bin/activate

2. Instalar dependencias:
   pip install -r requirements_hsemotion.txt

3. Primera ejecución (descarga modelos):
   python main_hsemotion.py


EJECUCIÓN
---------
python main_hsemotion.py


CONTROLES
---------
- 'q': Salir del programa
- 'd': Activar/desactivar detalles de todas las emociones


ESTADOS COGNITIVOS
------------------
FRUSTRADO: Anger, Contempt, Disgust, Sadness
DISTRAÍDO: Fear, Surprise  
ENTENDIENDO: Happiness
CONCENTRADO: Neutral


ARQUITECTURA
------------
1. face_detector.py
   - Detecta rostros usando Haar Cascade

2. hsemotion_classifier.py
   - Analiza emociones con EfficientNet preentrenado
   - Modelo: EfficientNet-B0 (state-of-the-art)

3. main_hsemotion.py
   - Integra detección + clasificación
   - Muestra resultados en tiempo real


VENTAJAS DE HSEMOTION
---------------------
✓ Ganador de competencias ABAW 2022-2024
✓ Estado del arte en precisión
✓ Velocidad de inferencia optimizada
✓ Detecta 8 emociones (incluye Contempt)

SOLUCIÓN DE PROBLEMAS
----------------------

Error: Lento en primera ejecución
  Normal - descarga modelos preentrenados

Error: "No se pudo acceder a la cámara"
  Cierra otros programas usando la webcam

Error con torch/timm
