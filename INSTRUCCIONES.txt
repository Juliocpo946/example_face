INSTRUCCIONES DE INSTALACIÓN Y EJECUCIÓN
=========================================

1. CREAR ENTORNO VIRTUAL
-------------------------
Windows:
  python -m venv venv
  venv\Scripts\activate
  
2. INSTALAR DEPENDENCIAS
-------------------------
pip install -r requirements.txt

NOTA: La primera vez que ejecutes el programa, transformers descargará
el modelo RoBERTa (aproximadamente 500MB). Esto puede tardar varios minutos
dependiendo de tu conexión a internet.


3. EJECUTAR EL PROGRAMA
-------------------------
python main.py


4. CONTROLES
-------------------------
- Presiona 'q' para salir del programa
- Presiona 'i' para activar/desactivar información detallada


5. ESTADOS COGNITIVOS DETECTADOS
---------------------------------
- CONCENTRADO: Rostro simétrico, características definidas
- DISTRAÍDO: Ojos parcialmente cerrados, confusión
- FRUSTRADO: Boca tensa, expresión asimétrica
- ENTENDIENDO: Expresión relajada, atención positiva


6. ARQUITECTURA DEL SISTEMA
----------------------------
1. face_detector.py: Detecta rostros usando Haar Cascade (OpenCV)
2. feature_extractor.py: Extrae 12 características geométricas del rostro
3. roberta_classifier.py: Convierte características a texto y usa RoBERTa
4. main.py: Integra todo y muestra resultados en tiempo real


7. SOLUCIÓN DE PROBLEMAS
-------------------------
Error: "No module named 'transformers'"
  Solución: pip install transformers torch

Error: "No se pudo acceder a la cámara"
  Solución: Verifica que ningún otro programa esté usando la cámara

Error de memoria al cargar RoBERTa
  Solución: Cierra otros programas y vuelve a intentar
