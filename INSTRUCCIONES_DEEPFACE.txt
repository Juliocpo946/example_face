SISTEMA DE ANÁLISIS COGNITIVO CON DEEPFACE
==========================================

INSTALACIÓN
-----------

1. Crear entorno virtual:
   Windows: python -m venv venv && venv\Scripts\activate
   Linux/Mac: python -m venv venv && source venv/bin/activate

2. Instalar dependencias:
   pip install -r requirements_deepface.txt

3. Primera ejecución (descarga modelos ~100MB):
   python main_deepface.py


EJECUCIÓN
---------
python main_deepface.py


CONTROLES
---------
- 'q': Salir del programa
- 'd': Activar/desactivar detalles de todas las emociones


ESTADOS COGNITIVOS
------------------
FRUSTRADO: angry, disgust, sad
DISTRAÍDO: fear, surprise  
ENTENDIENDO: happy
CONCENTRADO: neutral


ARQUITECTURA
------------
1. face_detector.py
   - Detecta rostros usando Haar Cascade

2. deepface_classifier.py
   - Analiza emociones con CNN preentrenada
   - Modelos: VGG-Face, FaceNet (97% precisión)

3. main_deepface.py
   - Integra detección + clasificación
   - Muestra resultados en tiempo real


VENTAJAS DE DEEPFACE
--------------------
✓ 97%+ precisión en reconocimiento facial
✓ Múltiples modelos state-of-the-art
✓ Detecta 7 emociones básicas
✓ Entrenado con millones de imágenes
✓ Usado en producción real


EMOCIONES DETECTADAS
--------------------
- angry: Enojado
- disgust: Disgusto/Asco
- fear: Miedo
- happy: Feliz
- sad: Triste
- surprise: Sorprendido
- neutral: Neutral


SOLUCIÓN DE PROBLEMAS
----------------------
Error: "No module named 'deepface'"
  pip install deepface

Error: Lento en primera ejecución
  Normal - descarga modelos preentrenados

Error: "No se pudo acceder a la cámara"
  Cierra otros programas usando la webcam

Error de TensorFlow
  pip install tensorflow==2.15.0
